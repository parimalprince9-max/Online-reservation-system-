ATM Management System

Project Overview

This project simulates an ATM system using Java and Object-Oriented Programming concepts.

Features

  • PIN Authentication
  • Balance Inquiry
  • Cash Withdrawal
  • Deposit Money
  • Mini Statement

OOP Concepts Used

  • Encapsulation
  • Inheritance
  • Polymorphism
  • Abstraction
